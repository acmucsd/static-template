## Getting Started
Install the dependencies
```bash
# Install Yarn:
$ git clone https://github.com/acmucsd/static-template
$ cd static-template
$ curl -o- -L https://yarnpkg.com/install.sh | bash
# Install the package dependencies
$ yarn
$ yarn start
```
